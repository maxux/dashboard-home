# Maxux Internal Dashboards Stack

Complete internal stack used for my personal dashboards providing realtime overview to a lot of useless
but absolutely mandatory metrics for my sanity.

There is no documentation, content is not intended to  be used outside this project but feel free to grab
some snippets if you find anything which suits you !
